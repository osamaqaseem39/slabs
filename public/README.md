# Public Folder

This folder contains static assets that will be served at the root URL path.

## Usage

Files placed in this folder can be referenced directly in your code:

- Images: `/image.png`
- Fonts: `/fonts/font.woff2`
- Icons: `/favicon.ico`
- Other assets: `/assets/file.pdf`

## Examples

```
public/
├── favicon.ico          # Site favicon
├── images/              # Image assets
│   ├── logo.png
│   └── hero-bg.jpg
├── fonts/               # Font files
│   └── custom-font.woff2
└── assets/              # Other static files
    └── resume.pdf
```
